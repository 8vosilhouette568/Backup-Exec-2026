## 🛡️ Best Practices for Data Protection

Follow these tips to keep your data safe:

*   Keep your backup files on a separate device from your main computer. If your computer hardware fails, your backup remains safe on the external drive.
*   Test your backups once a month. Try to restore one file to ensure the backup works correctly.
*   Use a strong password if you choose to encrypt your backup files. Keep this password in a safe place.
*   Update your software if a new version appears on the release page. Updates include fixes and performance changes.

## ❓ Frequently Asked Questions

**Does this software work on Windows 7?**
No, this version focuses on Windows 10 and Windows 11. 

**How do I restore my files?**
Open the application and click the "Restore" tab. Pick the backup job that contains your files. Select the files you want to get back and choose a location on your computer to save them.

**Can I run multiple backup jobs?**
Yes. You can create different jobs for different sets of files. For example, you can create one job for important work documents and a separate job for your family photo library.

**Does the software run in the background?**
Yes. The software runs as a service in the background so it can perform scheduled backups without interrupting your work. You can close the dashboard window without stopping the background service.
